Detailed Description

template<typename T>
class Page< T >

A preallocated chunck of memory.

Page Intended to loosely mimic the memory scheme of the operating system. Sets aside a chunk of memory, then allocates pieces within that chunk whenever we need it. Templated to allow for objects of any type, but all objects within a page must be the same type.
