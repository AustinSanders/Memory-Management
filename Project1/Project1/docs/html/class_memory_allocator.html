Detailed Description

template<typename T>
class MemoryAllocator< T >

A custom memory layer. Essentially a vector of equally sized pages.

MemoryAllocator Intended to loosely mimic the memory scheme of the operating system. Sets aside a chunk of memory, then allocates pieces within that chunk whenever we need it. Templated to allow for objects of any type, but all objects within the memory layer must be the same type.
